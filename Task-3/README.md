# CodeAlpha Task 3 - Data Visualization

## Objective
The objective of this project is to create visual representations of book data to identify patterns and insights.

## Dataset
Books dataset containing:
- Title
- Price
- Availability
- Rating

## Tools Used
- Python
- Pandas
- Matplotlib

## Visualizations Created
1. Price Distribution Histogram
2. Rating Distribution Bar Chart

## Key Insights
- Book prices vary significantly across the dataset.
- Rating distribution shows the frequency of ratings from One to Five.

## Output Files
- price_distribution.png
- rating_distribution.png
