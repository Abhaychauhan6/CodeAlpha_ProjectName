# CodeAlpha Task 2 - Exploratory Data Analysis (EDA)

## Objective

The objective of this project is to perform Exploratory Data Analysis (EDA) on a books dataset to understand data structure, identify trends, and extract meaningful insights.

## Dataset Information

The dataset contains information about books including:

* Title
* Price
* Availability
* Rating

## Tools Used

* Python
* Pandas

## Analysis Performed

1. Dataset Overview
2. Shape Analysis
3. Column Inspection
4. Missing Value Detection
5. Statistical Summary
6. Average Price Analysis
7. Maximum and Minimum Price Analysis
8. Rating Distribution Analysis

## Key Findings

* Total Books: 20
* Missing Values: 0
* Average Price: 38.05
* Highest Price: 57.25
* Lowest Price: 13.99
* Most Common Rating: One

## Conclusion

The dataset is clean and contains no missing values. Price analysis and rating distribution provide useful insights about the available books.
